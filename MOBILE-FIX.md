# Исправления для работы на мобильных устройствах Android

## Внесенные изменения:

### 1. Viewport и мета-теги
- Добавлен `maximum-scale=1.0, user-scalable=no` для предотвращения зума
- Добавлены мета-теги для PWA на iOS и Android
- Отключено автоопределение телефонных номеров

### 2. CSS оптимизация
- Добавлены touch-стили: `touch-action: manipulation`
- Убрана подсветка нажатий: `-webkit-tap-highlight-color: transparent`
- Улучшен скроллинг модальных окон: `-webkit-overflow-scrolling: touch`
- Размер шрифта в полях ввода увеличен до 16px (предотвращает зум на iOS)

### 3. JavaScript улучшения
- Добавлена поддержка touch событий для кнопок меню
- Улучшена инициализация Яндекс.Карт с задержкой для мобильных
- Добавлена обработка ошибок сети и офлайн режима
- Предотвращение скролла фона при открытии модальных окон

### 4. PWA улучшения
- Добавлены SVG иконки в manifest.json
- Улучшен Service Worker с логированием и обработкой ошибок
- Добавлено кэширование для офлайн работы

### 5. Карты Яндекс
- Добавлены мобильные настройки для карт
- Поддержка touch событий для выбора адреса
- Улучшена обработка ошибок загрузки карт

## Дополнительные рекомендации:

### Если проблемы остаются:

1. **Очистите кэш браузера** на Android устройстве
2. **Проверьте интернет соединение** - добавлен индикатор офлайн режима
3. **Обновите браузер** на Android устройстве
4. **Попробуйте другой браузер** (Chrome, Firefox, Samsung Internet)

### Для тестирования:
1. Откройте Chrome DevTools
2. Включите Device Mode (Ctrl+Shift+M)
3. Выберите Android устройство
4. Проверьте все функции

### Возможные проблемы:
- **Яндекс.Карты могут не работать** в некоторых браузерах - добавлен fallback
- **SMS верификация** может не работать в эмуляторах
- **Supabase соединение** может блокироваться некоторыми провайдерами

## Проверьте эти файлы:
- `index.html` - основная страница
- `login.html` - страница входа  
- `styles.css` - стили
- `script.js` - JavaScript логика
- `manifest.json` - PWA манифест
- `sw.js` - Service Worker

Все изменения направлены на улучшение работы на мобильных устройствах Android.
